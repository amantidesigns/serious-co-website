## Responsive Breakpoint Behavior

| Breakpoint | Grid Cols | Text Size | Button Layout |
|-----------|-----------|-----------|---------------|
| Mobile (< 640px) | 1 col | text-base | Stacked (100%) |
| Tablet (640-1024px) | 2-3 cols | text-lg | 2 per row |
| Desktop (> 1024px) | 3-4 cols | text-xl | 2 per row |

## Animation Flow

Each section inherits animation on:
1. **Hero**: Page load (initial → animate)
2. **All other sections**: Scroll into view (whileInView)

```
Timeline:
0ms ─────────────────────────────────────────
    Hero fades in

300ms ──────────────────────────────────────
      Hero content animates

1000ms+ ────────────────────────────────────
        User scrolls...
        Philosophy section: whileInView triggers
        Process section: whileInView triggers
        (Etc.)
```
